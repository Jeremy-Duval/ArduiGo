Digital Signal

A digital signal is a signal wich can take the value 0 or 1.

With this type of signal, you can use components whitch just can be in an 'on' or 'off' state.

An exemple : the DEL !

The component above is a DEL (light-emitting diode).

With this type of signal, you can use components whitch just can be in an 'on' or 'off' state.

In the code above, you begin to create a variable 'red_del' and affect it the value '2' (the pin wich is connected with the del). With this, you create a box in memory with the value '2' to remember the number of the del's pin and reuse it later.

Next, you use the 'pinMode' instruction. You give it 2 parameters : the number of the pin to initialise (your del's pin) and the state of the pin (here 'OUTPUT' because the power go to the del from the board ; 'INPUT' else).

Finally, you use the 'digitalWrite' instruction. It will send a low or high digital signal to a pin in function of parameters. The first is still the number of the pin wich you want to send the signal and the second is 'LOW' (to turn off the del) or HIGH' (to turn it on)
