What's inside this sample?
 

A CRCCard.vpp project which is the result of this sample.
 

How to use the sample?
 

1. Start VP-UML and create a CRCCard project.
 
 

2. Click CRC Card on the diagram toolbar and create a CRC Card. Edit the name as Vehicle.

 

 
 

3. Right-click on Attributes : and select Add -> Attribute to create an attribute on the CRCCard and edit the attribute name and type by double click on the attribute.

 
 
 

4. Let's create other cards, you will see the model are shown on the Model. And you can layout the cards by right-click on blank area and select Layout -> Cascade (or Tile).
 
 

5. Let's create the classes for the cards. Create a class diagram and select the models on Model. Drag the selected model into the class diagram, the classes are created automatically and they are shown on Model too.

 
   

Result
  The models are defined on CRCCard. And the CRCCard Models are converted into Class Models automatically.
