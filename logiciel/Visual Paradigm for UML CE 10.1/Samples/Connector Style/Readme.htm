What's inside this sample?
 

A sample project ConnectorStyle.vpp which contains one class diagram.
 

How to use the sample?
  1.

Start VP-UML and open the ConnectorStyle.vpp project. Open the Vehicle Type class diagram.
 
  2.

Right-click on the connector(genralization) bewteen PrivateCar and Vehicle and the popup menu appears. Select on the Connector Style and you will find that Oblique is selected
 
  3.

Try to move the connector. The connector draw a straight line bewteen its points.
 
  4.

Right-click on the connector and select Connector Style -> Rectilinear from the popup menu, the connector style is changed.
 
  5.

There are three connector styles to be selected:

Rectilinear

Oblique

Curve

Round Rectilinear
 
  6.

To select the connector style of all connectors in the diagram, right-click on the diagram and select Connector Style from the popup menu. (Different from the popup of a connector, there is no menu item is selected on diagram's popup menu because the diagram may contains different connectors with different connector styles.)
   

Result
 

Applying different connector styles on a connector changes the outlook of the connector.
