What's inside this sample?
 

A Game.dll is the library compiled from the C# code.
A Game.vpp is the result of this sample.

 

How to use the sample?
 

1. Start VP-UML click Tools -> Instant Reverse -> .NET dll or .exe files...
 
 

2. The dialog Instant Reverse Options appears and enter the path of the .NET dll or .exe that you want to be reversed. On this sample, select the Game.dll.
 
 

3. The .NET dll is reversed as UML model, and shown on Model.
 
 

4. Let's show the model on a class diagram. Right-click on package game on the Model. Select From Diagram -> Customize. Then select Show all levels in single diagram. Select all shapes of the diagrams by pressing CTRL+A and right-click on a class, select Related Model -> Show Relationships -> All to show the relationships of the classes. The class diagram is completed.

   

Result
  The UML model is reversed from the .NET dll or .exe.
