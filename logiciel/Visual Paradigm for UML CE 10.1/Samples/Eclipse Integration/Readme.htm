What's inside this sample?
 

A School directory which is the Eclipse Project directory finished the following sample (generate code from UML model).
A EclipseIntegration.vpp used to be imported for the Eclipse project on the following sample.
A School_Sample2 directory contains 2 source code files for the sample on reverse code to UML model
 

How to use the sample?
  This sample contains 2 parts: Install IDE Integration and Working with Eclipse Integration
 
Part I: Install IDE Integration
 

1. Start VP-UML, click Tools -> IDE Integration.... The VP Suite dialog box appears (It is used to install IDE Integration).
 
 

2. Select Eclipse/IBM WebSphere Studio Integration and click Next. You can choose other IDEs too, but on this sample, we show how to install Eclipse Integration only.

 
 

3. Enter the Eclipse Home Directory of the eclipse you want to install Eclipse Integration and click Next (installation will start automatically if the eclipse directory is valid).

 
 

4. Installing the Eclipse Integration into the eclipse, please wait while it is finished.
 
 

5. Installation of Eclipse Integration is finished, please click Finish to complete the installation.
 
Part II: Working with Eclipse Integration
 

1. Start the Eclipse which installed Eclipse Integration and create a Java Project named School and add a source folder named src.
 
 

2. After the project is created, right-click on School on Package Explorer and select VP-UML Project -> Import... to import the sample project (EclipseIntegration.vpp).
Please make sure you are right-click on the Package Explorer. If you cannot find the Package Explorer, please switch to Java Perspective first. If still not found, you can reset the perspective by Windows -> Reset Perspective.
 
 

3. The Import Project dialog appears and select the EclipseIntegration.vpp into the Source Project.
 
 

4. After the project is imported, VP-UML started up automatically. You will see a class diagram with 4 classes.
 
 

5. Click Sync to Eclipse on Diagram Navigator to generate the code from the UML model.
 
 

6. Eclipse will show on the front of the screen and the 4 classes is generated into the source folder.
   

Result
 

The code is updated according to UML model.

- Eclipse  - VP-UML

   

Tips & Trick
 

1. After create the project on Eclipse (or you can import the eclipse project from School_Sample2 by clicking File -> Import..., select Existing Project into Workspace and then select this sample's directory and select the project School_Sample2), you can create a new VP-UML Project instead of import a exisiting one. Right-click on the project on Package Explorer and select VP-UML Project -> New VP-UML Project.

 
 

2. Beside generating the code from UML Model, reversing the code into UML Model is also avaliable. Right-click on the project on Package Explorer and select Update VP-UML Model to reverse all the source code in the selected project.
