What's inside this sample?
 

A EJB folder contains the eclipse project which is the result of this sample.

 

How to use the sample?
 

There are 2 parts in this sample: Create EJB Diagram and Deploy EJB
 
Part I: Create EJB Diagram
 

1. Start Eclipse and create a Java Project. Right-click on the project on Package Explorer and select Open SDE-EC to start SDE-EC.
 
 

2. After SDE-EC is started, click File -> New Diagram -> New EJB Diagram to create a EJB diagram. Since EJB does not support the class under <default package>, EJB Diagram dialog appears and ask user for the package name. You see eJB on the folling image because the package name is come from the project name. We enter shop as the default package name of the EJB diagram.

 

 
 

3. EJB diagram is created, and we create an Entity Bean on it. Click Entity Bean on diagram toolbar and create the Entity Bean Item and add the attributes and functions on it.

 

 
 

4. Create the Primary Key Field of Item. Right-click on it and select Create Primary Key Field..., the CMP Field dialog appears.
 
 

5. Set the name to barcode and int as the type, and then click Finish. And create a find method on Item
 
 

6. Create a Session Bean Cart, click Session Bean on diagram toolbar and create the session bean. Add the operations on Cart

 

 
 

7. Right-click on the class, select Open Specification on popup menu. Select the tab page EJB Class Code Details and select all the Has remote, Has home, Has local and Has local home.

 
 

8. After create the EJB diagram, you can generate the source code from the models. Click Sync to Code on Diagram Navigator.

 
Part II: Deploy EJB
 

1. To deploy EJB, first set the Application Server. Click Modeling -> EJB -> Application Server Configuration... on menu bar.
 
 

2. After Application Server Configuration dialog appears. On this sample, select JBoss, and set the paths (Deploy Path, Client JAR Path, etc...). Click OK to finish.
 
 

3. Click Modeling -> EJB -> Deploy -> JBoss on menubar to deploy the EJB.
