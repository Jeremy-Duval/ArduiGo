What's inside this sample?
 

A ERD.vpp project which contains an Entity Relationship Diagram.
A ERD.dll file which contains SQL for this sample

 

How to use the sample?
 

This sample contains 2 Parts: Create ERD and generate DDL, Generate ORM (Object-Relational Mapping).

 
Part I: Create ERD and generate DLL
 

1. Start VP-UML and set the database for the project by clicking Tools -> Object-Relational Mapping (ORM) -> Database Configuration on menu bar.
 
 

2. The Database Configuration dialog appears and select the HSQL as the database. (A: You can select other database as you want. B: For this part of this sample, it is unnecessary to set the Database Setting, we just leave it blank.)
 
 

3. After select the database, select File -> New Diagram -> Others -> Entity Relationship Diagram to create an Entity Relationship Diagram (ERD).
 
 

4. Click Entity on diagram toolbar to create an Entity (Table) on the diagram. Edit the entity name to be Item.

 
 
 

5. To create the column on the entity Item, right-click on the entity and select New Column, edit the value of the column to be +barcode. A column is created as the primary key of the table ("+" means Primary Key). Alt+Shift+C is the hot key to create a column on an entity, or you can press Insert on a column to insert a new column.

  
 
 

6. You may find that, we havn't enter the data type of barcode. The default data type is integer , if you want to show the data type, right-click on the diagram and select Presentation Options -> Show Column Types

 
 
 

7. Create other columns and create an OrderLine entity. Select Item, click and drag the One-to-Many Relationship on the resource centric and edit the name to be OrderLine. The column Itembarcode is generated automatically as the Foreign Key.

 
 
 

8. Create the Order entity and add the Primary Key on it. Create the One-to-Many Relationship with Order and OrderLine, OrderLine will be added a Foreign Key automatically. And then set the 2 foreign keys of Order Line as its Primary Key and add the other columns on Order and Order Line. The ERD is completed.
 
 

9. To generate the DDL of the entities, select the entities on the diagram and right-click on them. Select Copy SQL -> Detail... to show the Generate SQL dialog and copy the content of Create.
 
Part II: Generate ORM (Object-Relational Mapping)
 

1. Open the project ERD.vpp (or use the project on Part I), right-click on the diagram and select Synchronize to Class Diagram (or use gesture, pressing right button on the diagram and move from right to left, draw a line.).
 
 

2. A class diagram is created with 3 classes (the classes are generated based on the entities).
 
 

3. Generate the code of the classes, click Tools -> Object-Relational-Mapping -> Generate Code....
 
 

4. Database Code Generationi dialog appears and enter the output path of the generated source code, click OK. (Because we havn't create a package for our classes, a dialog will be appears and ask for generating the source code on <default package>?
 
 

5. The sources are generated into the specified directory's src folder.
