What's inside this sample?
 

A SchoolForModeling.vpp project whichs contains two class diagrams.
 

How to use the sample?
 

1. Start VP-UML and open the HTMLReportGeneration.vpp project. Click Tools -> Report -> Generate HTML Report.... The Generate HTML dialog appears.
 

2. Enter the directory path where you want to report to be saved in the File location text field, and then select the diagrams in Print Diagrams to include the diagrams in the report .
 

3. Select the tab page Header/Footer to edit the header and footer of the report. After that, click Generate button to generate the report. After the report is generated, the default web browser will be launched for you to preview the report.
   

Result
 

A HTML report is generated with the selected diagrams, and the header and footer you specified (as shown in the figure below).
   

Tips & Trick
 

1. You may also choose to generate an HTML report using command line. For detailed instructions, please refer to the Report Generation chapter of the user's guide.
