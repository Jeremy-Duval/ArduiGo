What's inside this sample?
 

A SchoolForRose.mdl Rational Rose project which contains one use case diagram and two class diagrams. And a imported VP-UML Project ImportRationalRoseProject.vpp
 

How to use the sample?
 

1. Start VP-UML, select File -> Import -> Rose Project... on the menu bar. The Import Rose Option dialog box appears.
 
 

2. Select the file SchoolForRose.mdl by clicking on the ... button next to the field File path, and then select Model and diagram in Import mode. After that, click OK to import the Rose project.
 
 

3. After importing finished, one use case diagram and two class diagrams are created in VP-UML.
   

Result
 

The models and diagrams of the Rose project are imported into VP-UML.

   

Known Issue
 

1. The time required to import the Rose project depends on its file size. If the file is very large, it may need to take several minutes.
   

Tips & Trick
 

1. You can choose to import the model only by selecting Model only in Import mode. After importing the model you can use the Form Diagram function to generate diagrams from the models.
