What's inside this sample?
 

A Game Sources directory contains the java sources files to be reversed on this sample.
A Game Classes directory contains the java classes files to be reversed on this sample.
A GameSource.vpp is the result of this Java Source Instant Reverse.

 

How to use the sample?
 

This sample has 2 parts: Java Source Instant Reverse and Java Classes Instant Reverse
 
Part I: Java Source Instant Reverse
 

1. Start VP-UML click Tools -> Instant Reverse -> Java...
 
 

2. The dialog Instant Reverse Options appears and enter the directory of the sources that you want to be reversed. On this sample, Add Source Folder and then select the Game Sources directory.
 
 

3. The java source are reversed as Java Resources, and shown on Class Respository

 
 

4. Let's create the model on a class diagram. Right-click on package game under Java Resources. Select Reverse "game" to -> New Class Diagram. A new class diagram and model for selected element with children will be created

 
Part II: Java Classes Instant Reverse
 

1. Start VP-UML click Tools -> Instant Reverse -> Java...
 
 

2. The dialog Instant Reverse Option appears and Click on Add Class Folder to enter the directory of the classes or click on Add Jar to enter the path of a .jar file that you want to be reversed. On this sample, select the Game Classes directory.
 
 

3. The java classes are reversed as Java Resources, and shown on Class Respository

 
 

4. Let's create the model on a class diagram just like Java Source Instant Reverse (On Java Classes Instant Reverse, the parameters' name cannot be reversed.)

   

Result
  The UML model is reversed from the Java Source or Java Classes.
