What's inside this sample?
 

A Layout.vpp project which contains two class diagrams.
 

How to use the sample?
 

1. Start VP-UML and open the project Layout.vpp. There are two class diagrams as shown below:

i. Vehicle Structure

ii.Vehicle Type
 
 

2. Right-click on the empty space of Class Diagram1 and select Layout from the popup menu, the Layout dialog box appears.
 
 

3. There are six types of layout to select from, each has its own set of options.

Known Issue
 

1. Layout only supported in class diagram and collaboration diagram.
  2. Tree layout can only be applied when the diagram elements are organized in a tree structure (The child element has zero or one parent and the relationship cannot form a circle).
