What's inside this sample?
 

A OverviewDiagram.vpp project which contains one use case diagram and one class diagram.

 

How to use the sample?
 

1. Start VP-UML and open the OverviewDiagram.vpp project. Create an overview diagram by clicking File -> New Diagram -> Others -> Overview Diagram.
 
 

2. Click Diagram Overview and create a diagram overview on the overview diagram. Right-click on the diagram overview and select Associate to Diagram...

 

 
 

3. Select Diagram Dialog appears and select the Use Case Diagram Driving and click OK.
 
 

4. Create another diagram overview and click on Create New Diagram and select it button. And create a Sequence Diagram.
 
 

5. A sequence diagram is created and drag the models Driver and PrivateCar into the sequence diagram from the Model. Add a message bewteen the Driver and PrivateCar.

 

 
 

6. Activate the Overview Diagram1 again. Create one more diagram overview and associate with the class diagram Vehicle Structure. Add the relationship bewteent the diagram overviews.
   

Result
  Overview Diagram shows the relationships and dependencies bewteen diagrams.
