What's inside this sample?
 

A project PinningConnectors.vpp which contains two class diagrams.
 

How to use the sample?
 

1. Start VP-UML and open the PinningConnectors.vpp project, open the class diagram Vehicle Type (as shown in the figure below).
 
 

2. Try to move the note. The connection end points of the anchor (the connector in dotted-line) are not fixed.
 
 

3. Click on the anchor to select it, you will see the Pin resource buttons appear at both ends of the anchor.
 
 

4. Click the Pin button that is near the class GearBox to fix this connection end point.
 
 

5. Drag the connection end point near GearBox so that it points at the attribute gearType (to indicate the note is for this attribute).
 
 

6. Move the note again, you will see that the connection end point near GearBox is fixed to the attribute gearType.
   

Result
 

The position of a pinned connection end point will be fixed no matter how the connecting shapes move.
