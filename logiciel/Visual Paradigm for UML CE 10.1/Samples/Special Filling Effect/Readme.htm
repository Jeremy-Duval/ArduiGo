What's inside this sample?
 

A SpecialFillingEffect.vpp project which contains one class diagram.
 

How to use the sample?
 

1. Start VP-UML and open the SpecialFillingEffect.vpp project. Open Vehicle Structure (as shown below).
 
 

2. Select all the classes and then right-click on one of them and select Format -> Fill... or activate the Property (left-bottom of the image below) and click the ... button on the Fill property to edit the fill color of the selected classes
 
 

3. When the Format Fill Color dialog appears, select Gradient in Fill style to select a gradient fill color.
 
 

4. You can change the Transparency, select colors by selecting a set of colors or specify the custom color for the fill color.
 

5. After you change the color, you can click Add to Themes... and enter the name of the new theme to add the color into the Themes. (Gradient Style will not be remembered into the theme)
 
 

6. And then select the theme you want to use and click Apply or OK. On this sample, we use the theme Violet and with the the first Gradient Style as the color of classes.
 
 

7. Select the package and set its fill color to Mint Blue The whole diagram's fill colors are set well
   

Result
 

The selected classes will be filled with the new fill color.
