What's inside this sample?
 

A SchoolForDesign.vpp project which contains one use case diagram.
 

How to use the sample?
 

1. Start VP-UML and open the SchoolForDesign.vpp project. Open Enrollment System as shown below.
 
 

2. Right-click on the use case Enroll into a course and select Use Case Details... from the popup menu. The Use Case Details - Enroll into a course appears.
 
 

3. Enter the use case description as shown below.
 
 

4. Let's preview the use case description in the PDF report. Click Tools -> Report -> Generate PDF Report.... The Generate PDF dialog box appears.
 
 

5. Select Enrollment System in Print Diagrams to include it in the report, and then click on the Preview button to preview the PDF report.
 
 

6. Click on the Next Page button several times until you see the use case description of the use case Enroll into a course (as shown below).

   

Result
  The use case description is specified for the use case Enroll into a course.
