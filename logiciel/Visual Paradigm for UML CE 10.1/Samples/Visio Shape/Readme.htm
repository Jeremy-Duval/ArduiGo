What's inside this sample?
 

A VisioSample.vpp file which contains one class diagram.
 

How to use the sample?
 

1. Start VP-UML and open the VisioShape.vpp project. Open class diagram Visio Sample as shown below:
 
 

2. Click on the Visio Shape button on the diagram toolbar, and then click on the empty space of the diagram to create a Visio Shape.

 
 
 

3. Double click on the text "Visio Shape" to edit the name to be Sample 3
 
 

4. Double click on the shape Sample 3 to edit it. The Visio® will be started.
 
 

5. Enter Traffic on Search for Shapes. After searching, select the Speed limit icon and drag it into the pane. The shape is created and resize it. Finally, close the Visio®.
 
 

6. The image of the shape is shown on VPUML now. Repeat step 5 and draw a Bus Stop. Select Generic Connector button on the diagram toolbar, create the generic connector bewteen the 2 visio shapes.
   

Result
 

The class diagram which is drawn using Visio Shapes is completed as shown below.
   

Known Issu
  1. In order to use Visio Shape, you need to have Microsoft® .NET Framework 1.1 and Microsoft® Office Visio® 2003 installed. It is important that you MUST install the .NET Framework before installing Visio®.
