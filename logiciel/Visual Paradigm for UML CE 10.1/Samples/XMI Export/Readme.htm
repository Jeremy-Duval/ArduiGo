What's inside this sample?
 

A XMIExport.vpp project which contains one one class diagram.
A vehicle.xml xml file exported by this sample.

 

How to use the sample?
  1. Start VP-UML and open the XMIExport.vpp sample project. And click File -> Export -> XMI... on the menu bar. The Export XMI dialog box appears.

 
 

2. Enter the Output Path of the exported XMI, and then click OK to start export.
   
Result
 

A XMI file is generated to the specified location, which contains the models and diagrams information of the project.
   

Known Issue
 

1. VP-UML supports the exporting of XMI of the following versions:

- XMI 2.1 for UML 1.4 (OMG)
 
 

2. The time required to export the XMI file depends on the size of the project. If the project is very large, it may take several minutes to export. In this case, close other programs before performing the export.
