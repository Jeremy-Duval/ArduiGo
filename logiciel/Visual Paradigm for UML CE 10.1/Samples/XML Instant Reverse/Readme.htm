What's inside this sample?
 

A Shop.xml is the XML file contains the data which will be used to reverse on this sample.
A Shop.vpp is the result of this sample.

 

How to use the sample?
 

1. Start VP-UML click Tools -> Instant Reverse -> XML...
 
 

2. The dialog Instant Reverse Options appears and enter the path of the XML file that you want to be reversed. On this sample, select the Shop.xml.
 
 

3. The XML is reversed as UML model, and shown on Model.
 
 

4. Let's show the model on a class diagram. Selec on elements on the Model and right-click on them, select Form Diagram -> Navigation to form the class diagram.

   

Result
  The UML model is reversed from the XML file.
